Estimated Time: 30 minutes

Objective: Learn regular verbs to ask for and to see for restaurant and sightseeing scenarios.

1. Intro Video Placeholder (You)

Explain that to ask for (e.g., I ask for a beer) and to see (e.g., I see the beach) are regular -ir verbs, like to live. Highlight their use for ordering food and describing sights. Preview restaurant nouns.

2. Grammar Focus – To Ask For and To See

Structure: Subject + verb + article + noun (+ adjective). Examples: I ask for a cold beer → Pido una comida fría, You see a pretty square → Ves una plaza bonita. Key Rule: Both follow regular -ir endings (-o, -es, -e, -imos, -en), but to ask for has a stem change (e → i) in all forms except we. Task: Change I ask for a salad to You see a museum.

4. Listening Prompt

Say: I ask for a coffee. I see a pretty mountain. Ask: What do you ask for? → Pause for: I ask for a dessert. Goal: Recognize to ask for, to see, and nouns.

5. Reading + Matching Task

Match: I ask for a new dessert →, I see a pretty church →, It’s a big table →. Identify: subject, verb, noun in You ask for a cold beer.

6. Speaking Prompt

You: What do you ask for? → Student: I ask for a beer.

You: “Say you see a landscape.” → I see a pretty landscape.

You: What is that? (point to church) → It’s a church.

7. Writing Practice

  • I ask for a dessert → Pido un postre.
  • You see a big church → Ves una iglesia grande.
  • We ask for a table → Pedimos una mesa.

8. Recap Challenge (Speaking)

You: “Ask for a coffee.” → I ask for a coffee.

You: “Say what you see.” → I see a pretty square.
