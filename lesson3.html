Day 3: There is/are + Gender Practice + More Travel Nouns

Estimated Time: 25 minutes

Objective: Use hay (there is/are) to describe surroundings, reinforce article gender, and add travel nouns.

1. Intro Video Placeholder (You)

  • Explain hay for describing what’s around (e.g., There is a beach).
  • Recap article + noun + adjective structure.
  • Preview new travel nouns.

2. Grammar Focus – There is/are

Hay = “There is / There are” (same for singular/plural).

Structure: Hay + article + noun (+ adjective)

  • Hay un hotel grande. – There is a big hotel.
  • Hay unas playas bonitas. – There are some pretty beaches.

Key Rule: Articles match noun gender/number; hay doesn’t change.

Task: Change “There is a new suitcase” to “There are some ugly taxis.”

4. Listening Prompt (You Provide)

Transcript:

  • Say: “There is a big city. There is a pretty market.”
  • Ask: “What is that?” → Pause for: “It’s a new shop.”

Goal: Recognize hay and nouns.

5. Reading + Matching Task

  • Match: There is a new station, There is a big market, It’s a pretty city
  • Identify: article, noun, adjective in “There are some pretty shops.”

6. Speaking Prompt

  • You say: “There is a big city.” → Student: Hay una ciudad grande.
  • You say: “What is that?” (point to shop) → “It’s a pretty shop.”
  • You say: “Describe a market.” → “There is a big market.”

7. Writing Practice

Translate:

  • There is a new shop. → Hay una tienda nueva.
  • There are pretty stations. → Hay unas estaciones bonitas.
  • What is that? It’s a big airport. → ¿Qué es eso? Es un aeropuerto grande.

8. Recap Challenge (Speaking)

  • You say: “A pretty shop” → Student: A pretty shop.
  • You say: “There are big markets.” → Student: There are big markets.
